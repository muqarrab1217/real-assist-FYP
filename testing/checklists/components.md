Shared Components Checklist
===========================

Tables/Lists
------------
- Pagination, sorting, and filtering work together; empty/loading/error states render correctly.
- Row selection and bulk actions reflect current data; responsive columns truncate gracefully.

Modals/Drawers
--------------
- Open/close via trigger, escape, and outside click; focus trap enforced; scroll locked.
- Primary/secondary actions behave; destructive actions confirm.

Toasts/Alerts
-------------
- Appear for success/error; auto-dismiss timing correct; stack behaves.
- Screen readers announce messages.

Date/Time Pickers
-----------------
- Min/max constraints honored; manual input validated; timezone/locale formatting correct.

File Upload
-----------
- Accepts allowed types/sizes; rejects others with clear errors; progress and cancel work; multiple files behave.

Search/Autocomplete
-------------------
- Debounced queries; keyboard selection; no-results state; handles network errors gracefully.
